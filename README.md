## 📌 Featured Projects

### [Quant War Room](https://github.com/agatafietko/quant-war-room)
Live trading dashboard built on Streamlit. Implements Bollinger Band, RSI, and Mean Reversion strategies with real-time signal generation and portfolio tracking.

### [Squeeze Oracle](https://github.com/agatafietko/squeeze-oracle)
Short interest intelligence app that surfaces squeeze candidates using multi-factor scoring across float, short interest, volume, and momentum signals.

### [Gies Sustainability Dashboard](https://github.com/agatafietko/sustainability-dashboard)
React 19 / TypeScript dashboard mapping 2,022 Gies College publications to all 17 UN SDGs using an LLM + FAISS semantic pipeline and Gemini AI integration. Deployed at [agatafietko.github.io/sustainability-dashboard](https://agatafietko.github.io/sustainability-dashboard).

### Short-Alpha Intelligence Pod
Multi-agent system for short-side equity research built in Antigravity IDE. Agents handle data ingestion, signal generation, and synthesis in parallel.

### Bosch B2B Portal Analytics
Graduate consulting project with Bosch Power Tools. Built user segmentation models, cart-to-order funnel dashboards, and association rule analysis on browsing behavior to support portal personalization.

### Illinois Statistics Datathon (Synchrony Financial)
Intraday call center volume disaggregation across four portfolios using a LightGBM + Prophet ensemble pipeline on Databricks / AWS. Top competitive placement.
